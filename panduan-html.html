Pengenalan & Struktur Dasar

HTML (HyperText Markup Language) bukan bahasa pemrograman — ini bahasa markup yang ngasih struktur ke halaman web. Browser baca HTML terus nampilin teks, gambar, form, dan lainnya. CSS ngatur tampilan, JavaScript ngatur interaksi — tapi fondasinya selalu HTML dulu.

💡 File HTML disimpan dengan ekstensi .html. Buka pakai browser apa aja (Chrome, Firefox, Edge) — tidak perlu compile kayak C atau Python.

Setiap dokumen HTML punya kerangka standar: deklarasi tipe dokumen, elemen <html>, lalu <head> (info halaman) dan <body> (konten yang kelihatan).

index.html
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Halaman Pertamaku</title>
</head>
<body>
  <h1>Halo, Web!</h1>
  <p>Ini halaman HTML pertamaku.</p>
</body>
</html>

2

Tag Penting — Teks & Heading

Tag paling sering dipakai buat konten teks. Heading h1–h6 bikin hierarki judul (h1 = paling penting, cuma satu per halaman idealnya). Tag p, strong, em, dan br ngatur paragraf & penekanan.

teks.html
<h1>Judul Utama</h1>
<h2>Subjudul</h2>
<p>Paragraf biasa. Bisa panjang beberapa kalimat.</p>
<p>Teks <strong>tebal penting</strong> dan <em>miring penekanan</em>.</p>
<p>Baris pertama<br>Baris kedua (pakai br)</p>
<hr>
<blockquote>Kutipan dari sumber lain.</blockquote>
<code>console.log()</code> <!-- kode inline -->
<pre>Blok kode
beberapa baris</pre>

Link, Gambar & List

HyperText = teks yang bisa di-link. Tag <a> bikin hyperlink, <img> sisipkan gambar (self-closing), dan list pakai ul/ol + li.

media.html
<!-- Link -->
<a href="https://developer.mozilla.org" target="_blank" rel="noopener">
  Buka MDN (tab baru)
</a>

<a href="#bagian-bawah">Loncat ke bawah</a>

<!-- Gambar -->
<img
  src="https://picsum.photos/200/120"
  alt="Foto contoh random"
  width="200"
  height="120"
>

<!-- List -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<ol>
  <li>Belajar struktur</li>
  <li>Tambah styling</li>
  <li>Tambah interaksi</li>
</ol>

<p id="bagian-bawah">Anchor target di sini.</p>

Atribut, div/span & Block vs Inline

Atribut ngasih info tambahan ke tag: id (unik), class (bisa dipakai berkali-kali), data-* buat data custom. div & span adalah wadah generik — div block-level, span inline.

layout.html
<div id="kartu" class="box highlight" data-topik="html">
  <h3>Kartu Konten</h3>
  <p>Paragraf di dalam <span class="label">inline span</span>.</p>
</div>

<div class="box">
  <p>div lain — class sama, id beda (atau tanpa id).</p>
</div>

<!-- Inline style (jarang dipakai di proyek besar) -->
<p style="color: #2563eb; font-weight: 600;">
  Teks biru pakai style inline
</p>

5

Form — Input dari User

Form dipakai login, daftar, search, contact — apapun yang butuh input user. Bungkus dengan <form>, label dengan <label>, dan berbagai <input> sesuai kebutuhan.

form.html
<form action="/kirim" method="post">

  <label for="nama">Nama:</label>
  <input type="text" id="nama" name="nama" placeholder="Nama lengkap" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <label for="pwd">Password:</label>
  <input type="password" id="pwd" name="password" minlength="8">

  <label for="bio">Bio:</label>
  <textarea id="bio" name="bio" rows="3"></textarea>

  <fieldset>
    <legend>Level</legend>
    <label><input type="radio" name="level" value="pemula" checked>  Pemula</label>
    <label><input type="radio" name="level" value="menengah">  Menengah</label>
  </fieldset>

  <label><input type="checkbox" name="setuju" required>  Setuju syarat</label>

  <label for="bahasa">Bahasa:</label>
  <select id="bahasa" name="bahasa">
    <option value="html">HTML</option>
    <option value="css">CSS</option>
    <option value="js">JavaScript</option>
  </select>

  <button type="submit">Kirim</button>
  <button type="reset">Reset</button>

</form>

6

Tabel

Tabel untuk data tabular — jadwal, harga, perbandingan. Struktur: table → thead/tbody → baris tr → sel th (header) atau td (data).

tabel.html
<table>
  <caption>Nilai Ujian</caption>
  <thead>
    <tr>
      <th scope="col">Nama</th>
      <th scope="col">Matematika</th>
      <th scope="col">IPA</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Budi</th>
      <td>85</td>
      <td>90</td>
    </tr>
    <tr>
      <th scope="row">Ani</th>
      <td>92</td>
      <td>88</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="3">Rata-rata kelas: 89</td>
    </tr>
  </tfoot>
</table>

scopecol = header kolom, row = header baris. Screen reader bisa baca tabel dengan benar.
colspanGabungkan sel horizontal. Ada juga rowspan buat vertikal.
captionJudul tabel — langsung di bawah tag table. Bagus buat aksesibilitas & konteks.
💡 Jangan pakai tabel cuma buat layout halaman — itu zaman old-school. Layout pakai CSS (flexbox/grid). Tabel = data tabular aja.
7

Semantic HTML

Daripada semua pakai div, HTML5 punya tag yang berarti: header, nav, main, article, footer, dll. Browser & mesin pencari paham struktur halaman — lebih baik buat SEO dan aksesibilitas.

semantic.html
<header>
  <h1>Blog Bro</h1>
  <nav>
    <a href="#">Beranda</a>
    <a href="#">Artikel</a>
  </nav>
</header>

<main>
  <article>
    <header>
      <h2>Belajar HTML</h2>
      <p><time datetime="2025-06-04">4 Juni 2025</time></p>
    </header>
    <p>Konten artikel di sini...</p>
    <footer>Oleh: Bro</footer>
  </article>

  <aside>
    <h3>Terkait</h3>
    <ul><li>Panduan CSS</li></ul>
  </aside>
</main>

<footer>
  <p>© 2025 Belajar Pemrograman Bro</p>
</footer>

Tips & Trik HTML

Beberapa kebiasaan bagus biar HTML-mu rapi, valid, dan siap dilanjut ke CSS & JavaScript.

tips.html
<!-- Komentar HTML -->
<!-- Tidak dirender, berguna catatan atau matikan sementara -->

<!-- Karakter khusus (escape) -->
<p>5 < 10 dan A & B — pakai &lt; &amp; atau tulis di teks biasa</p>

<!-- Void elements (tidak perlu penutup) -->
<img src="foto.jpg" alt="...">
<br>
<input type="text">
<meta charset="UTF-8">

<!-- Link CSS & JS (biasanya di head / sebelum </body>) -->
<link rel="stylesheet" href="style.css">
<script src="app.js" defer></script>
ValidasiCek HTML di validator.w3.org — tangkap tag tidak tertutup atau struktur salah.
IndentasiRapikan nested tag — lebih gampang debug. Editor kayak VS Code auto-format.
Langkah berikutSetelah paham HTML, lanjut CSS buat tampilan & JavaScript buat interaksi — ketiganya biasanya dipisah file: index.html, style.css, app.js.
🚀 HTML = struktur. Jangan obsesi styling di HTML (font, center lawas sudah deprecated). Taruh visual di CSS — separation of concerns.
